Alternative Output Types

Aside from the standard .cif files (which contain the crystal structure information), MatterGen produces several other useful outputs that help you understand and evaluate the generated materials. These include:

Extended XYZ Files (.extxyz)

A single file that stores multiple generated structures as sequential frames. This format is useful for visualizing or postâ€‘processing the full set of atomic coordinates and compositions.

Denoising Trajectories (generated_trajectories.zip)

A ZIP archive containing the denoising process for each generated structure. These files contain intermediate snapshots (often in .extxyz format) that show the progressive refinement of a noisy input into a stable structure. Such trajectories are valuable for analyzing the convergence behavior of the diffusion process.

Evaluation Metrics File (metrics.json)

When running the evaluation script (e.g., using MatterSim), the output is stored in a JSON file. This file typically contains:

  • Stability Metrics: Percentage of generated structures that fall within a threshold energy above the convex hull.
  • Novelty Metrics: Fraction or count of generated structures that do not match any structure in a reference dataset (using a disordered structure matcher).
  • Uniqueness Metrics: Number or percentage of structures that are unique among the generated samples.
  • Geometric Accuracy: Average root-mean-square distance (RMSD) between the generated structures and their DFT-relaxed counterparts.
  • Property Predictions: Optionally, values of properties such as band gap, magnetic density, or bulk modulus predicted using surrogate ML models.
  • Other Statistical Summaries: Distributions (mean, median, standard deviation) of key properties across the generated dataset.

Visualization and Plot Files (External)

Although MatterGen itself outputs raw data formats (like .cif and .extxyz), you can leverage external tools and libraries for further analysis:

  • 2D Graphs, Charts, and Plots: Use libraries like Matplotlib, Seaborn, or Plotly to generate plots (e.g., histograms of property distributions, scatter plots comparing generated vs. reference properties, convergence curves of the diffusion process).
  • Statistical Analysis Scripts: Custom Python scripts can analyze the metrics file, perform hypothesis testing, or generate interactive dashboards using libraries like Dash or Bokeh.
  • Visualization with VESTA: While VESTA is an external standalone application for crystal structure visualization, you can write custom scripts that automatically launch VESTA to view .cif files.
