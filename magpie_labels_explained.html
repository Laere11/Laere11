Magpie Property Labels Explained

This page provides a reference table describing the most common Magpie property labels (such as “Number,” “MendeleevNumber,” “AtomicWeight,” etc.) and the statistical prefixes (e.g., “minimum,” “maximum,” “range,” “mean,” “avg_dev,” “mode”) that you might encounter in the Matminer Magpie feature set.

Each row in the table explains both the underlying property and the meaning of the statistical label for non-experts. The goal is to help you understand the significance of each descriptor as it applies to your material's composition (e.g., NaCl).

Label Definition / Significance
Number

Base Property: The atomic number of each element (i.e., number of protons in the nucleus).

Statistical Labels:

  • minimum Number: The smallest atomic number among the elements in the composition.
  • maximum Number: The largest atomic number among the elements.
  • range Number: The difference between the largest and smallest atomic number.
  • mean Number: The average atomic number of all elements.
  • avg_dev Number: The average absolute deviation from the mean atomic number.
  • mode Number: The most frequently occurring atomic number in the composition.
MendeleevNumber

Base Property: An index used by Dmitri Mendeleev in his periodic table ordering. It’s a historical or alternative numbering system that can reflect chemical trends more smoothly than pure atomic number.

Statistical Labels:

  • minimum MendeleevNumber … mode MendeleevNumber have analogous meanings, giving you min, max, range, mean, average deviation, and the most common Mendeleev number among the elements in the composition.
AtomicWeight

Base Property: The standard (average) atomic weight of each element (in atomic mass units).

Statistical Labels:

  • minimum AtomicWeight, maximum AtomicWeight, etc. follow the same pattern, capturing the min, max, range, mean, avg deviation, and mode of atomic weights for the composition’s elements.
MeltingT

Base Property: The melting temperature (in K) of each element (if known).

Statistical Labels:

  • minimum MeltingT, maximum MeltingT, range MeltingT, mean MeltingT, avg_dev MeltingT, mode MeltingT — these describe how the melting points of the constituent elements vary in the mixture.
CovalentRadius

Base Property: The approximate radius of an atom when covalently bonded, measured in angstroms (Å).

Statistical Labels:

  • minimum CovalentRadius, maximum CovalentRadius, range CovalentRadius, mean CovalentRadius, avg_dev CovalentRadius, mode CovalentRadius describe the distribution of covalent radii among the elements in the composition.
Electronegativity

Base Property: A measure of how strongly each element’s atom attracts electrons in a chemical bond (often using the Pauling or Allen scale).

Statistical Labels:

  • minimum Electronegativity, maximum Electronegativity, range Electronegativity, mean Electronegativity, avg_dev Electronegativity, mode Electronegativity provide the spread and central tendencies of electronegativities in the composition.
NsValence / NpValence / NdValence

Base Property: The count of valence electrons in the s, p, or d orbitals for each element. These are sometimes referred to as NsValence, NpValence, and NdValence in Magpie’s data.

Statistical Labels:

  • minimum NsValence … mode NsValence, etc. for the p and d orbitals as well. They summarize how the valence electron counts in s/p/d orbitals vary among the elements in the composition.
row

Base Property: The row (period) of the periodic table in which each element resides. Rows are counted top to bottom (e.g., H is row 1, Li is row 2, etc.).

Statistical Labels:

  • minimum row, maximum row, range row, mean row, avg_dev row, mode row describe how the elements in your composition span different rows of the periodic table.
column

Base Property: The column (group) of the periodic table in which each element resides. Columns are typically counted left to right (e.g., group 1 for alkali metals, group 17 for halogens, etc.).

Statistical Labels:

  • minimum column, maximum column, range column, mean column, avg_dev column, mode column give the spread of columns for the composition’s elements.

Interpreting the Statistics

minimum / maximum: The smallest or largest value of a given property among all elements in your composition.

range: The difference between the maximum and minimum values. It tells you how widely the property varies across the elements.

mean: The arithmetic average of the property’s values among the elements in the composition.

avg_dev: The average absolute deviation from the mean, reflecting how spread out the values are around that mean.

mode: The most frequently occurring value among the elements in the composition (often most relevant if multiple elements share the same discrete property).

Why These Properties Matter

  • Atomic Number & MendeleevNumber: Indicate where the element sits in the periodic table, often correlating with chemical reactivity and bonding characteristics.
  • AtomicWeight & MeltingT: Provide insight into the mass and thermal behavior of each element. A composition with high average melting point, for instance, might produce a material with high thermal stability.
  • CovalentRadius & Electronegativity: Strongly influence bonding and structure formation in the crystal. Larger radii or big differences in electronegativity often lead to specific types of bonding arrangements.
  • NsValence / NpValence / NdValence: Show how many electrons are available in each orbital, guiding predictions about oxidation states and bonding preferences.
  • row & column: Summarize how your composition spans the periodic table’s rows (periods) and columns (groups), which often determines how “mixed” or “similar” the constituent elements are from a chemical standpoint.

By examining these properties and their statistical descriptors, you can gauge how your multi‑element composition (like NaCl) spans key chemical and physical attributes, which in turn can hint at the resulting material’s stability, bonding type, and potential applications.
